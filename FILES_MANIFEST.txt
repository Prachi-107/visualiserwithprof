📞 QUICK REFERENCE
================================================================================

WHAT GETS CAPTURED:
  • Layer name & type
  • Total execution time (ms)
  • Forward pass time (ms)
  • Backward pass time (ms)
  • Weight update time (ms)

OUTPUT FORMAT:
  # NNTrainer Layer Execution Timing Data
  layer_name layer_type total_time forward backward update
  input Input 0.002 0.002 0 0
  conv2d_1 Conv2D 5.430 2.100 3.330 0

HOW COLORS WORK:
  • Green  (<0.5ms)   = Fast, no optimization needed
  • Yellow (0.5-2ms)  = Moderate, potential improvements
  • Red    (>2ms)     = Slow, likely bottleneck
